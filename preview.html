Screens Features

App Features

  • Order Tracking: Real-time status from "Order Received" through "Delivered"
  • Progress Visualization: 1 of 5 step progress indicator
  • Mobile-First Design: iOS-optimized with SwiftUI, clean interface
  • Push Notifications: Built-in support for order updates
  • Dark Mode Support: iOS system colors with smooth transitions
  • Real-Time Updates: Estimated delivery times that update as order progresses
  • Admin Dashboard: Separate management interface for order management
Screen Descriptions

Screen Flow

  • Home Screen: Order number input → "Track" button → Redirects to tracker page
  • Tracker Page: 5-step progress indicator (Order Received → Preparing → Out for Delivery → Arriving → Delivered). Shows order details, customer info, items, and estimated time.
  • Push Notifications: Visual indicator in status bar shows when new updates arrive
  • Admin Dashboard: Overview of all active orders, delivery statistics, and status management interface
